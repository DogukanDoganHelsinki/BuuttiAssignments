Random API 2

GET /random
Accepts query parameters
  • min (number): minimum value (inclusive). Default 0.
  • max (number): maximum value (exclusive). Default 100.
  • float (boolean): if set to true, a decimal number is provided. Default false.
Example:
/random?min=-100&float=true
Returns a decimal number between -100 (inclusive) and 100 (exclusive)
